### Code Examples

Since I write about engineering, expect lots of code blocks with syntax highlighting, line numbers, and a copy button:

```typescript
// Here's a simple example, notice the copy button in the top-right.
interface User {
  id: string;
  name: string;
  email: string;
}

async function fetchUser(id: string): Promise<User> {
  const response = await fetch(`/api/users/${id}`);
  return response.json();
}
```

The syntax highlighting adjusts based on the language (TypeScript, Python, Rust, etc.), making code easier to read.
