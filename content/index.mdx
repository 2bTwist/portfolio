# Welcome to My Portfolio

This is an MDX-powered portfolio with a recursive file tree navigation system.

## Features

- 🚀 **Next.js 16** with App Router
- 📝 **MDX Support** for rich content authoring
- 🌲 **Recursive File Tree** that mirrors your content structure
- 🎨 **Tailwind CSS** for styling
- 🌙 **Dark Mode Ready** with next-themes

## Navigation

Use the sidebar to explore:
- **Blog posts** - Technical articles and insights
- **Projects** - Portfolio projects and case studies
- **About** - Learn more about me

---

Start by clicking on any file in the sidebar! 👈
